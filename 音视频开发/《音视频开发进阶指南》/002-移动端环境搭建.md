# 移动端环境搭建



## 交叉编译原理



| 名称     | 作用                                                         |
| -------- | ------------------------------------------------------------ |
| CC       | 编译器，c源码到汇编文件                                      |
| AS       | 汇编器，汇编文件到机器码                                     |
| AR       | 打包器，从一个库中删除或者增加目标代码模块                   |
| LD       | 链接器，为生成的目标代码分配地址空间，将多个目标文件链接成一个库或可执行文件 |
| GDB/LLDB | 调试工具                                                     |
| STRIP    | 以最终生成的可执行文件或库文件作为输入，消除掉其中源码       |
| NM       | 查看静态库或动态库的符号表                                   |
| objdump  | 查看静态库或动态库的方法签名                                 |
