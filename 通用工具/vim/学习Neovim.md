# Neovim安装与配置架构总览

## 安装Nerd fonts
Nerd fonts 本身并不是一种新的字体，而是把常用图标以打补丁的方式打到了常用字体上。

比如我在 VSCode 里最常用的是 Fira Code 字体，那么我就要去安装这个打了 Nerd fonts 补丁的 FiraCode 字体。或者你也可以到[官网](https://www.nerdfonts.com/font-downloads)找到你喜欢的字体。
